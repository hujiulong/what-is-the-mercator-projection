## 什么是墨卡托投影
地球是一个球体，但是通常我们会把世界画在一个矩形平面上，得到一副地图。一个球体并不能直接展开成一个矩形，所以在这个展开的过程中会做一些变形，墨卡托投影就是一种最常用的“展开方式”，我们平时使用的纸质地图，还有百度地图，高德地图这种电子地图都是使用墨卡托投影。
